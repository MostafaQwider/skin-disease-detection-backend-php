# Skin Disease Detection Backend (PHP)

A pure PHP backend API for a medical mobile application focused on skin disease detection using AI-powered image analysis.

The system connects doctors and patients in a unified healthcare platform, enabling doctors to create clinics, provide diagnoses, and manage medical consultations, while patients can explore clinics, receive AI-assisted skin disease detection, and book appointments.

---

## Overview

This backend serves as the core of a dermatology-focused healthcare application that integrates artificial intelligence with real medical services.  
It provides structured and secure communication between doctors and patients while supporting automated skin disease diagnosis through image analysis.

---

## Key Features

- AI-based skin disease detection using image analysis
- Doctors can create and manage dermatology clinics
- Doctors can perform and store medical diagnoses
- Patients can browse available clinics
- Patients can view diagnosis results
- Appointment booking between patients and clinics
- RESTful API with JSON responses
- Optimized for mobile application integration

---

## Technologies

- Pure PHP (no framework)
- RESTful API
- JSON-based communication
- Image upload handling
- External AI/ML model integration

---

## AI Diagnosis

Patients or doctors can upload skin images through the application.  
The AI model analyzes the images and returns:
- Predicted skin disease
- Confidence score

The backend processes and delivers these results in a clean and reliable JSON format.
